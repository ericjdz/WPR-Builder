Template-driven DOCX generator

Weekly Progress Report builder

Generate Microsoft Word weekly progress reports by injecting your values into the original DOCX template XML. The template stays structurally intact, including headers and media, while the browser fills token placeholders and repacks the final .docx.

Template Embedded base64 DOCX, no fetch required
Engine JSZip + direct XML token substitution
Layout Original form preserved, 5 days × 3 tasks
Input Form editor and raw JSON textarea
Multi-week reports reuse the same template body and insert a page break between generated weeks before the final DOCX download.

Form editor

Edit one or more weeks visually. Every week always renders 5 day groups with 3 task rows per day in the template.

Ready.

JSON input

Paste JSON directly, validate it, then load it back into the form or generate the DOCX from the textarea.

Required week fields: department, weekNumber, internName, company, deptDeployed, supervisorName, fromDate, toDate, hoursThisWeek, totalHoursCompleted, totalRequiredHours, workSchedule, preparedDate, supervisorDate. Empty task rows are allowed.

  • Each week is normalized to exactly 5 days.
  • Each day is normalized to exactly 3 task rows.
  • Special XML characters are escaped before token injection.
  • Generation fails fast if required fields are missing.
